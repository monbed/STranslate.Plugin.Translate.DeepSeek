# STranslate.Plugin.Translate.DeepSeek

一个为 [STranslate](https://github.com/ZGGSONG/STranslate) 应用提供的 DeepSeek 翻译插件。

## 快速开始

### 安装

1. 下载最新版本的.spkg 插件包
2. 打开 STranslate 软件，进入 设置 > 插件。将下载的文件拖入窗口即可安装。

### 配置

首次使用前需要进行如下配置：

1. 打开 STranslate 应用
2. 添加 DeepSeek 作为翻译引擎
3. 配置必要参数：
   - **API Key**：DeepSeek API 密钥（从 [DeepSeek 官网](https://www.deepseek.com) 获取）
   - **模型选择**：选择要使用的 DeepSeek 模型
   - **其他参数**：根据需求调整温度等参数
